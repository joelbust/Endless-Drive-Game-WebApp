# Endless Drive Game Web App

A browser-based 3D endless driving game built using JavaScript and Three.js.  
The project features real-time rendering, interactive gameplay mechanics, dynamic scene management, and WebGL-based graphics running directly in the browser.

## Live Demo

[Play the Game Here](https://joelbust.github.io/Endless-Drive-Game-WebApp)

---

## Features

- 3D endless driving gameplay
- Real-time rendering with Three.js and WebGL
- Interactive camera and scene management
- Dynamic lighting and shadows
- Browser-based gameplay with no installation required
- Responsive rendering optimized for different screen sizes

## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/joelbust/Endless-Drive-Game-WebApp.git
```

Navigate into the project directory:

```bash
cd Endless-Drive-Game-WebApp/Final-Project
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Gameplay

The player controls a vehicle in an endless driving environment while the scene continuously renders and updates in real time.  
The project focuses on combining interactive gameplay with browser-based 3D graphics and rendering optimization.
